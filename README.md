# Genome

## Auteurs
Rémy Delmas & Frane Mboumba

## Description

Le logiciel permet d'effectuer des traitements sur des données représentant le génome humain dans un temps le plus court possible.

Il met en oeuvre une plateforme distribuée de traitement des séquences ADN.
